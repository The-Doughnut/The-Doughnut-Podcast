Adobe Flash Player

Your version of Flash:

The Flash Player plugin is essential if you want to view this website properly. If you do not see any animation on this webpage, it means that you do not have Flash installed. Please click here to get a Flash Player emulator plugin called Ruffle.

As of now, Ruffle is still in development and many Flash content may not be viewed properly with it. Eventually it will become stable, but as of now if you absolutely want to view Flash content without bugs you may download an old Flash Player version if security risks are no concern.

Below are old versions of Flash Player.
DISCLAIMER: I am not responsible for security vulnerabilities caused by old Flash Player plugins. I guess use a virtual machine?
Flash Player version 32.0.0.334
Flash Player version 29.0.0.113
Flash Player version 28.0.0.161


If you are good with tech, you can also modify your current version of Flash Player if the kill-switch has already been activated. On page 47 of Adobe's documentation of Flash Player 32, you can figure out that EnableAllowList=1 can be added to the mms.cfg file, which may allow you to disable the kill-switch. I have not tested it out myself, so I do not know how well it works. If you need additional help, you can read more of the PDF.

One more thing: your web browser. Chances are, your web browser dropped Flash support in January 2021. So if you are going to download an older version of Flash Player, you should download an older version of your browser to have Flash support, but beware that you will not have the latest security updates.
