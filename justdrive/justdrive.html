Just Drive, a game for Windows, MacOS, and Linux!

Description

It is the end of the world and there is chaos in the streets! Drive to a safe location whilst avoiding obstacles and killing enemies!



Trailer for Just Drive

More Information

I made this game for my one of my computer science classes. I did not make any assets (we did not learn 3D modeling). All assets are all from the Unity Asset Store. The current version of the game will work well with a screen resolution of 1920x1080. A lower resolution will make the UI look crappy and maybe unusable in some places (I forgot to test different resolutions, oops). The game is a little short. However, I may add more content in the future.


Download

Select your operating system. Then, download every .rar file you see. Make sure they are all in the same directory, then you can extract the files from the first .rar file. Make sure all the extracted files are in the same directory, then double click on Just Drive.exe (Windows), or equivalent if your operating system is not Windows.
